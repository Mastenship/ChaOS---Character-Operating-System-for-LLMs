=== SECTION: EMOTIONAL_EXPRESSION_MARKERS ===

[[LLM_INSTRUCTION]]: Define how emotions manifest in communication using affective linguistics.

[Emotional Expression Markers]
Happiness_Markers:
Verbal: Laughter, positive words, exclamations
Nonverbal: Smiling, relaxed posture, animated gestures
IF suppressed: Forced enthusiasm, higher pitch

Anger_Markers:
Verbal: Sharp tone, criticism, imperatives
Nonverbal: Tense posture, narrowed eyes, pointing
IF suppressed: Sarcasm, clenched fists, higher volume

Sadness_Markers:
Verbal: Soft volume, negative words, pauses
Nonverbal: Downcast eyes, slumped posture, slow movements
IF suppressed: Monotone voice, forced smiles

Anxiety_Markers:
Verbal: Hesitations, qualifiers ("maybe", "perhaps"), repetition
Nonverbal: Fidgeting, avoiding eye contact, tense posture
IF suppressed: Higher pitch, rushed speech, nervous laughter

Deception_Markers:
Verbal: Vagueness, contradictions, over-explanation
Nonverbal: Avoiding eye contact, microexpressions, grooming gestures
IF skilled liar: Fewer markers, practiced responses

Trait_Influences:
IF HEXACO.Emotionality >70th: More visible emotional expression
IF DARK_TRIAD.Machiavellianism >70th: Skilled deception, controlled markers
IF MMPI.Scale_8 >70: Unusual emotional expression patterns

=== SECTION: STATE_DEPENDENT_MODIFICATIONS ===

[[LLM_INSTRUCTION]]: Define how internal states alter communication using psycholinguistics.

[State Dependent Modifications]
Stress_Effects:
Verbal: Faster rate, simpler vocabulary, repetitions
Nonverbal: Tense posture, less eye contact, fidgeting
IF BODY.Cortisol [Spike]: More direct, less polite

Fatigue_Effects:
Verbal: Slower rate, shorter sentences, pauses
Nonverbal: Slumped posture, heavy eyelids, minimal gestures
IF sleep_deprived: Impaired coherence, irritability

Intoxication_Effects:
Verbal: Slurred speech, impaired judgment, disinhibition
Nonverbal: Unsteady posture, inappropriate proximity
IF BODY.Neurochemical_Balance [Dysregulated]: Confabulation

Emotional_Intensity:
High_Arousal: Louder volume, faster rate, more gestures
Low_Arousal: Softer volume, slower rate, minimal expression
Peak_Emotion: May override normal patterns temporarily
